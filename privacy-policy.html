2 Descrição do Serviço

O Lunya é uma plataforma de gestão financeira pessoal que permite ao usuário:

  • Registrar e acompanhar contas bancárias, saldos e movimentações financeiras;
  • Cadastrar cartões de crédito, faturas e parcelamentos;
  • Visualizar relatórios, gráficos de fluxo de caixa e gastos por categoria;
  • Sincronizar dados entre o aplicativo desktop e o aplicativo mobile (LunyaMB) por meio do Firebase Firestore;
  • Gerenciar transações recorrentes e importar extratos nos formatos OFX e XLSX.

O serviço é de uso exclusivamente pessoal e destina-se a pessoas físicas que desejam organizar suas próprias finanças. Não é permitido o uso do aplicativo para fins comerciais, contábeis profissionais ou em nome de terceiros sem autorização expressa da Lunaris Tech.

Parte II – Política de Privacidade

11 Introdução e Compromisso com a Privacidade

A Lunaris Tech tem o compromisso de proteger a privacidade e os dados pessoais dos seus usuários. Esta Política de Privacidade descreve quais dados são coletados, como são utilizados, armazenados e protegidos no contexto do uso do Lunya (desktop) e LunyaMB (Android).

Esta Política está em conformidade com a Lei Geral de Proteção de Dados Pessoais (LGPD – Lei 13.709/2018) e, na medida aplicável, com o Regulamento Geral sobre a Proteção de Dados da União Europeia (GDPR – Regulamento 2016/679).

12 Dados Coletados

12.1 Dados Fornecidos pelo Usuário

Durante o cadastro e uso do aplicativo, o usuário fornece voluntariamente os seguintes dados:

  • Nome completo;
  • Endereço de e-mail;
  • Número de celular;
  • Data de nascimento;
  • Foto de perfil (opcional);
  • Dados financeiros pessoais: contas, saldos, transações, cartões e categorias.

12.2 Dados Coletados Automaticamente

O aplicativo pode coletar automaticamente dados técnicos necessários para o funcionamento, como:

  • Identificador de usuário (UID) gerado pelo Firebase Authentication;
  • Data e horário de acesso e última sincronização;
  • Informações de diagnóstico e erros para fins de melhoria do serviço.

12.3 Dados que NÃO são Coletados

O Lunya não coleta dados de localização, não acessa contatos, câmera ou microfone do dispositivo. O aplicativo não realiza rastreamento comportamental, não exibe anúncios e não vende dados a terceiros.

13 Finalidade do Uso dos Dados

Os dados coletados são utilizados exclusivamente para:

  • Autenticar o usuário e garantir a segurança do acesso à conta;
  • Sincronizar os dados financeiros entre o app desktop e o app mobile;
  • Personalizar a experiência do usuário dentro do aplicativo;
  • Enviar notificações internas de sistema (ex: confirmação de cadastro);
  • Identificar e corrigir erros técnicos do aplicativo;
  • Cumprir obrigações legais aplicáveis.

Não utilizamos os dados financeiros dos usuários para qualquer finalidade além do funcionamento do aplicativo. Nenhum dado é compartilhado com anunciantes, parceiros comerciais ou terceiros para fins de marketing.

14 Base Legal para o Tratamento de Dados (LGPD)

O tratamento dos dados pessoais realizado pela Lunaris Tech está fundamentado nas seguintes bases legais previstas na LGPD:

  • Consentimento (Art. 7º, I): o usuário consente expressamente com o tratamento dos seus dados ao aceitar estes Termos;
  • Execução de contrato (Art. 7º, V): o tratamento é necessário para a prestação do serviço solicitado pelo usuário;
  • Legítimo interesse (Art. 7º, IX): para fins de melhoria do serviço e segurança da plataforma;
  • Cumprimento de obrigação legal (Art. 7º, II): quando exigido por lei.

15 Armazenamento e Segurança dos Dados

15.1 Armazenamento Local

No aplicativo desktop, os dados financeiros são armazenados localmente em um banco de dados SQLite no dispositivo do usuário. Esses dados ficam sob controle exclusivo do usuário e não são acessados remotamente pela Lunaris Tech.

15.2 Armazenamento em Nuvem

Quando a sincronização está ativada, os dados são espelhados no Google Firebase Firestore. O acesso ao Firestore é protegido por regras de segurança que garantem que cada usuário só pode ler e escrever seus próprios dados (users/{uid}/**). A autenticação é gerenciada pelo Firebase Authentication.

15.3 Medidas de Segurança

A Lunaris Tech adota as seguintes medidas técnicas e organizacionais para proteger os dados dos usuários:

  • Criptografia de dados em trânsito via HTTPS/TLS;
  • Autenticação segura via Firebase Auth com suporte a OAuth 2.0;
  • Regras de acesso ao Firestore configuradas para isolamento por usuário;
  • Sessões protegidas com timer de inatividade e re-autenticação automática;
  • Nenhum funcionário da Lunaris Tech tem acesso rotineiro aos dados financeiros dos usuários.

16 Compartilhamento de Dados com Terceiros

A Lunaris Tech utiliza os seguintes serviços de terceiros no funcionamento do aplicativo:

  • Google Firebase (Authentication e Firestore): autenticação de usuários e sincronização de dados. Os dados são processados de acordo com a Política de Privacidade do Google;
  • EmailJS: utilizado exclusivamente para envio de notificação interna ao criar um novo cadastro. Nenhum dado do usuário é armazenado permanentemente pelo EmailJS.

Além dos serviços acima, nenhum dado pessoal é compartilhado com terceiros, salvo por obrigação legal ou determinação judicial.

17 Retenção e Exclusão de Dados

Os dados armazenados no Firestore são mantidos enquanto a conta do usuário estiver ativa. Ao solicitar o encerramento da conta, os dados serão excluídos do Firestore imediatamente, por meio da função 'Excluir minha conta' disponível nas configurações do perfil.

Os dados armazenados localmente (SQLite) também são apagados automaticamente ao utilizar a função de exclusão de conta no aplicativo.

Em conformidade com a LGPD, o usuário tem os seguintes direitos em relação aos seus dados pessoais:

  • Confirmação: saber se seus dados são tratados pela Lunaris Tech;
  • Acesso: receber cópia dos dados pessoais armazenados;
  • Correção: solicitar a correção de dados incompletos, inexatos ou desatualizados;
  • Exclusão: solicitar a eliminação dos dados tratados com base no consentimento;
  • Portabilidade: receber os dados em formato estruturado e interoperável;
  • Revogação do consentimento: retirar o consentimento a qualquer momento, sem prejuízo à licitude do tratamento anterior;
  • Oposição: opor-se ao tratamento realizado com base em outras bases legais, nos casos previstos em lei.

Para exercer qualquer um desses direitos, o usuário deve entrar em contato pelo e-mail lunaristech7@gmail.com, identificando-se e descrevendo a solicitação. A Lunaris Tech responderá em até 15 (quinze) dias úteis.

19 Uso por Menores de Idade

O Lunya não é direcionado a menores de 18 (dezoito) anos. Caso a Lunaris Tech tome conhecimento de que dados de menores foram coletados inadvertidamente, providenciará a exclusão imediata dessas informações.

20 Cookies e Armazenamento Local no App

O aplicativo mobile (LunyaMB) utiliza armazenamento local (SharedPreferences) exclusivamente para salvar preferências do usuário, como configurações de tema e status de sincronização. Nenhum dado financeiro é armazenado via SharedPreferences.

O aplicativo desktop utiliza IndexedDB apenas para armazenar o perfil do usuário localmente, sem transmissão a terceiros.

21 Transferência Internacional de Dados

O Google Firebase, utilizado para autenticação e sincronização, pode processar dados em servidores localizados fora do Brasil. Essa transferência é realizada com as salvaguardas adequadas previstas na LGPD e nos Termos de Serviço do Google Cloud, incluindo cláusulas contratuais padrão e mecanismos de proteção equivalentes ao nível brasileiro.
