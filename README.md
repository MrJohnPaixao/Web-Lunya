# Lunya Page

Página pública da Lunya, produto da Lunaris Tech para finanças pessoais.

Repositório: `MrJohnPaixao/Web-Lunya`

## Estrutura

- `index.html` - página principal.
- `privacy-policy.html` - página pública de Termos de Uso e Política de Privacidade.
- `lunya-page.css` - estilos da página.
- `lunya_brand_assets/` - ícones e imagens usados pela página.
- `downloads/` - arquivos públicos de download.
- `Lunya_Termos_Privacidade_v1.0.docx` - documento legal original.

## Downloads Disponíveis

### Windows Desktop

- Instalador EXE: `downloads/windows/Lunya_1.0.0_x64-setup.exe`
- Instalador MSI: `downloads/windows/Lunya_1.0.0_x64_en-US.msi`

### Android APK

- `downloads/app-release.apk`
- `downloads/app-arm64-v8a-release.apk`

## Instalador Windows

A pasta `downloads/windows/` contém os instaladores públicos usados pelos botões da página. A pasta `installer/` fica reservada para scripts e recursos NSIS, caso sejam versionados junto com a página.

## Publicação no GitHub Pages

Este projeto é estático. Para publicar:

1. Envie o conteúdo para um repositório GitHub.
2. Ative GitHub Pages usando a branch principal e a raiz do projeto.
3. Confirme que os arquivos de download existem nos caminhos usados pelo `index.html`.

## Observação

Os APKs e instaladores Windows são versionados neste repositório para permitir download direto pela página.
